# NovelFrame MVP 確認チェックリスト

## 概要
NovelFrame MVP版の完成度を確認するための詳細なチェックリストです。Phase 2 Beta版に進む前に、すべての項目を確認し、必要な修正を完了させてください。

## 1. アプリケーション基盤

### 1.1 起動・終了
- [ ] アプリケーションが正常に起動する
- [ ] 起動時間が5秒以内
- [ ] 起動時にクラッシュしない
- [ ] 終了時にメモリリークがない
- [ ] 終了時に一時ファイルがクリーンアップされる
- [ ] 複数起動時の動作が適切

### 1.2 基本UI
- [ ] メインウィンドウが正しく表示される
- [ ] ウィンドウサイズ変更が正常に動作
- [ ] 最小化/最大化が正常に動作
- [ ] 各パネルが適切に配置されている
- [ ] レスポンシブデザインが機能している
- [ ] ダークモード対応（将来対応予定）

### 1.3 パフォーマンス
- [ ] メモリ使用量が500MB以下
- [ ] CPU使用率が待機時10%以下
- [ ] レンダリングが60FPS以上
- [ ] 30分以上の連続使用でクラッシュしない
